# Hello World Django App

This simple Django application demonstrates two endpoints:

## How to Run

1. **Clone the Repository** (if not already done):

git clone <url>
cd helloworld


2. **Install Django** (if not already installed):

pip install django



3. **Run the Server**:

python manage.py runserver





After starting the server, you can access the application endpoints:

- **JSON Response**: Access at `http://127.0.0.1:8000/hello/` to receive a JSON response containing `{"Message": "Hello World!"}`.
- **HTML Response**: Visit `http://127.0.0.1:8000/hellohtml/` to view a page displaying "Hello World!" in bold.
